Looking at this, I need a delocal expression that equals `n_orig` at the end. After the loop: `a = a_orig + n_orig` and `b = b_orig - n_orig`, so `a - b = (a_orig - b_orig) + 2*n_orig`. If `a_orig == b_orig` (as in both test cases), then `(a - b) / 2 = n_orig`.

```
procedure xJk2Q9_step(int n, int a, int b)
    local int t = n
    from n = t do
        skip
    loop
        a += 1
        b -= 1
        n -= 1
    until n = 0
    delocal int t = (a - b) / 2
```
